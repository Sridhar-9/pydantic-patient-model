# Patient Data Validation System (Pydantic)

A clean and modular implementation of data validation using Pydantic, simulating a real-world patient record system.

---

## Features

-  Strong data validation using Pydantic
-  Custom field validators (email domain restriction)
-  Model-level validation (age-based emergency contact rule)
-  Nested models (Address)
-  Computed fields (BMI calculation)
-  Type-safe schema design

---

## What This Project Demonstrates

This project showcases:

- Schema design using Pydantic
- Data integrity enforcement
- Clean project structuring
- Real-world validation scenarios
